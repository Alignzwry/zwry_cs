# zwry_cs
Counterstrike 2 External Kernel Cheat

# Usage:
compile driver <br />
compile start.exe (you can also use create.cmd every time you recompile the driver if you dont want start.exe) <br />
run zwry_esp.exe <br />
use delete.cmd to unload the driver <br />
